# Tarun R — Personal Portfolio

A personal portfolio website built with a glassmorphism aesthetic, featuring an AI-powered bio section that generates dynamic introductions using the Claude API.

🌐 **Live Demo:** [iam-tarun-86-.github.io/tarun-portfolio](https://iam-tarun-86-.github.io/tarun-portfolio)

---

## Features

- **Glassmorphism UI** — frosted glass cards, soft gradient backgrounds, smooth hover effects
- **AI-generated bio** — Claude Sonnet API generates a dynamic personal intro on every load with a regenerate option
- **Responsive layout** — works across desktop and mobile
- **Sections** — About, Skills, Projects, Certifications, Contact

## Tech Stack

- Vanilla HTML, CSS, JavaScript (no frameworks)
- [Claude Sonnet API](https://www.anthropic.com) — AI bio generation
- GitHub Pages — hosting

## Featured Project

### Local AI Security Log Analyzer
A full-stack SIEM automation tool powered by local LLMs. Uses a 3-node LangGraph agentic workflow (Ingest → Classify → Router) for real-time forensic log analysis — fully offline, zero external API costs, 100% data privacy.
